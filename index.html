Платформа для талантов

Подберите формат участия под свои цели

Мы разработали два тарифа — для авторов проектов и тех, кто хочет поддерживать талантливых ребят. Каждый тариф открывает доступ к личному кабинету, событиям и сообществу.

Как посмотреть страницу локально?

  1. Установите зависимости командой npm install.
  2. Запустите локальный сервер: npm run dev.
  3. Откройте http://localhost:3000 в браузере.
